OutSync

Project

OutSync is a game about duality, it's a Die and Retry game in VR. This game is playing with the rythm of the music by changing the lights, and weapons.

Personal Feedback

This project was my first experience with VR, both as a player and a developer. Participating in this game jam was a very enriching experience, giving me the opportunity to learn how to use ProBuilder on Unity, an essential tool for Level Design on Unity.

In addition, this project was my first experience as a full-time Level Designer on a project. My main role was to create 2D schematics and then implement them on Unity in 3D. This responsibility allowed me to explore new skills and actively contribute to the design of the game's levels.
